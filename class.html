Web Design Fundamentals

This fundamental course will introduce students to basic web based application and techniques used in the design of standard web pages. This is an introductory web design course and will focus our learning how to write valid XHTML & CSS.

Course Outline

Major modules, themes, or topics.

  • Introduction to Web Design
  • Investigate & Apply basic HTML Structure
  • Investigate & Apply Text Formatting
  • Apply Hyperlinks
  • Create & Apply CSS Style Sheets
  • Apply Folder Structure & File Management
  • Apply Useability and Accessibility Techniques
  • Utilize FTP to Publish to the Web
